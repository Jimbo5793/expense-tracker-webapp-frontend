# 💰Frontend for Expense Tracker

A high-performance React application engineered to simplify group finances. This platform eliminates the awkwardness of "who owes what" by automating debt resolution, splitting bills equitably, and providing deep insights into collective spending habits.

## 🚀 Key Capabilities

### 💎 **The User Experience**

- **Intelligent Settlement**: Proprietary logic ensures 1:1 debt resolution without "over-payment" bugs.
- **Dynamic Group Logic**: Seamlessly add or remove participants with real-time balance recalibration.
- **Intuitive Architecture**: A card-centric design optimized for quick entry and clear visibility on any screen size.
- **Smart Categorization**: Tag expenses to see exactly where the group’s money is going.

### 📊 **Analytics & Insights**

- **Interactive Visuals**: Powered by Chart.js, offering granular breakdowns of spending by category.
- **History at a Glance**: A searchable, filterable ledger of every transaction, from grocery runs to rent payments.
- **Real-time Statistics**: Instant summaries of total group spend vs. individual contributions.

### 📊 **Data Visualization**
- Pie charts for spending breakdown by category
- Interactive data filtering
- Summary statistics and insights
- Export capabilities (future feature)

### 🔧 **Technical Features**
- Built with React 19.2.3 and modern hooks
- React Router for seamless navigation
- Chart.js for beautiful data visualization
- Context API for state management
- Progressive Web App capabilities

## 🚀 Tech Stack

- **Frontend**: React 19.2.3
- **Routing**: React Router DOM 7.12.0
- **Charts**: Chart.js 4.5.1 & React Chart.js 2 5.3.1
- **Styling**: CSS3 with modern features
- **State Management**: React Context API
- **Build Tool**: Create React App
- **Testing**: Jest & React Testing Library

## 📋 Prerequisites

Before running this application, make sure you have:
- Node.js (version 14 or higher)
- npm or yarn package manager

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jimbo5793/expense-tracker-webapp-frontend.git
   cd expense-tracker-webapp-frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

The app will automatically reload when you make changes to the code.
